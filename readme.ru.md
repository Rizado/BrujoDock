# BrujoDock, v. 26.2

**Универсальный док для Linux** — быстрый, настраиваемый, на основе плагинов.

Выпущен 22.02.26

## Системные зависимости

### Debian/Ubuntu/Mint

```bash
sudo apt update
sudo apt install python3-gi python3-gi-cairo python3-pytzdata gir1.2-gtk-3.0\
    gir1.2-wnck-3.0 gir1.2-gnomedesktop-3.0 libcairo2-dev
```

### Fedora

```bash
sudo dnf install python3-gobject python3-gobject-cairo python3-pytzdata \
    gtk3 libwnck3 cairo-devel
```

### Arch Linux

```bash
sudo pacman -S python-gobject python-cairo python-pytzdata gtk3 libwnck cairo
```

### OpenSUSE

```bash
sudo zypper install python3-gobject python3-gobject-cairo python3-pytzdata \
    gtk3 libwnck-3-0 cairo-devel
```

## Проверка зависимостей

```bash
python3 -c "from gi.repository import Gtk, Wnck; import cairo, pytz; print('Все зависимости найдены')"
```

## Языки

- Английский (English) 
- Русский
- Испанский (Español)

Смена: Настройки → Технические → Язык

- **Внимание!** Логирование поддерживает только английский язык.

## Настройки

Ctrl + правая кнопка мыши на доке → "Настройки"

| Вкладка         | Что настраивается                            |
|-----------------|----------------------------------------------|
| **Основные**    | Высота дока, отступы, радиус углов           |
| **Плагины**     | Включение/отключение плагинов, их сортировка |
| **Технические** | Язык, режим логирования                      |

## Плагины

Сейчас док включает 4 плагина:

| Плагин             | Описание                                                |
|--------------------|---------------------------------------------------------|
| **Icon Panel**     | Закреплённые приложения, иконки запущенных приложений   |
| **Clock**          | Время, дата, несколько часовых поясов                   |
| **SysMon**         | Исопльзование процессора, памяти, температура процессора |
| **Battery Status** | Уровень заряда батареи, статус зарядки                  |

## Решение проблем

| Проблема                               | Решение                                                                |
|----------------------------------------|------------------------------------------------------------------------|
| **Док не запускается**                 | Проверьте зависимости: ```python3 -c "from gi.repository import Gtk``` |
| **Иконки не отображаются**             | Проверьте, установлены ли приложения                                   |
| **Часы показывают неправильное время** | Проверьте часовой пояс системы                                         |
| **Плагин не работает**                 | Включите в настройках, вкладка "Плагины"                               |

## Известные проблемы

**Когда вы закрепляете иконку AIMP, иногда требуется перезапуск дока для её отображения.**

AIMP для Linux сейчас на стадии альфа-тестирования. Дождитесь хотя бы релиз-кандидата.
